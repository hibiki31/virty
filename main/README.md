## Git運用メモ
ローカルDevelopでこまめにコミット、スカッシュして正式なメッセージをつける。

```
Fix バグ修正
Add 機能追加
Update バグではない修正やドキュメントの修正
Remove 不要なコードの削除
Revert 変更取り消し
```

マージするときは以下の規則で行う。

```
Merge branch 'develop' v1.0.0
```
